-   React koristi Create react app koji brzinski postavi prostor za izradu apliakcije, ali under the hood to je zapravo webpack
-   ja pišem kod u src, a u dist će se buildati svi static assets
-   kod reacta se napiše npm build i onda on sve stavi u folder build, ovo je zapravo ista ta stvar

-   u terminalu prvo napravim package.json file s:
npm init -y

- da se sadržaj iz src kopira u dist
npm run build

- da se otvori live web server
npm run dev
